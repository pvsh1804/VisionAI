# VisionFlow AI

AI-Powered Intelligent Traffic Management & Smart Mobility Operations Platform.

## Overview

VisionFlow AI is an enterprise-style Intelligent Traffic Management System built using YOLOv8, Flask, OpenCV, EasyOCR, and a modern Smart Mobility dashboard.

It is designed to demonstrate real-time traffic intelligence, adaptive signal control, emergency vehicle priority, incident management, traffic prediction, digital twin visualization, and real-time heat map density overlay.

## Features

- Real-time vehicle detection using YOLOv8
- Vehicle classification: car, bus, truck, motorcycle, bicycle
- Weighted congestion intelligence
- Adaptive traffic signal recommendation
- AI decision engine with explainable recommendation
- Traffic prediction and trend detection
- OCR-based license plate/text detection
- Emergency vehicle priority mode
- Accident / road blockage alert mode
- Enterprise Smart Mobility dashboard
- Live 2D Digital Twin intersection
- Real-time heat map traffic density overlay
- Camera and AI health monitoring
- Event timeline for operator workflow
